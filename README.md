# 趣味提案アプリ

自分の特性(インドア/アウトドア・社交性・体力・創作志向・学習志向)と、
予算・使える時間・お住まいの都道府県を診断すると、マッチ度つきでオススメの趣味を
提案するWebアプリです。バックエンドや外部APIを持たない、完全静的なSPAです。

## 主な機能

- **特性診断**: 10問の選択式アンケート(うち2〜3問は「子供の頃、時間を忘れて夢中に
  なったのは?」のような内省質問)。回答から5軸のプロファイルを算出します。
- **タイプ診断**: 5軸のプロファイルから「あなたは○○タイプ」を表示します。
- **趣味提案**: 48件の趣味データベースから、5軸のマンハッタン距離をもとにした
  マッチ度(0〜100%)で上位6件を提案します。予算・時間の超過はハードフィルタでは
  なく減点として扱い、注意文つきで候補に残します。
- **今日から始められる情報**: 初期投資レンジと道具、月額目安、無料/約5,000円/
  本格スタートの3段階スターターキット、年間費用シミュレータ、始め方の3ステップを
  各趣味カードに表示します。
- **地域コミュニティリンク**: 選択した都道府県をもとに、ジモティー・こくちーずプロ・
  Google マップ・X(旧Twitter)の検索リンクを自動生成します。
- **お気に入り・診断結果の保存**: ブラウザの localStorage に保存します。読み出し時は
  JSON の構文エラーだけでなく「構文は正しいが型が期待と違う」破損データ(数値・null・
  必須フィールド欠落など)も型ガードで検証し、不正ならデフォルト値にフォールバック
  するため、壊れたデータが残っていてもアプリはクラッシュしません。

## 技術構成

- Vite + React 18 + TypeScript(完全静的SPA)
- 状態管理: React の useState のみ
- スタイル: 素のCSS(CSS変数によるテーマ、レスポンシブ、`prefers-color-scheme`
  によるダークモード対応)
- テスト: Vitest(推薦エンジン・コミュニティリンク生成・データ整合性・タイプ診断・
  費用シミュレータなどをユニットテストで担保)

## セットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

## テストの実行

```bash
npm test
```

## ビルド

```bash
npm run build
```

## ディレクトリ構成

```

  data/hobbies.ts       # 趣味DB(48件)
  data/questions.ts     # 診断質問(内省質問を含む10問)
  data/regions.ts       # 47都道府県
  lib/recommend.ts      # 推薦エンジン(純関数)
  lib/community.ts      # コミュニティリンク生成(純関数)
  lib/storage.ts        # localStorage ラッパ(バリデータつきで防御的)
  lib/savedDiagnosis.ts # 保存された診断データの型ガード
  lib/deriveType.ts     # タイプ名診断(純関数)
  lib/cost.ts            # 年間費用シミュレータ(純関数)
  lib/profile.ts         # 回答から5軸プロファイルを算出(純関数)
  components/Questionnaire.tsx
  components/ResultList.tsx
  components/HobbyCard.tsx
  App.tsx / main.tsx / index.css
  **/*.test.ts           # Vitest によるユニットテスト
```
